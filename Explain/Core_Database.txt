CLASSE : Database (Core)
FICHIER : src/Core/Database.php

DESCRIPTION :
Gère la connexion à la base de données MySQL. Elle utilise le pattern Singleton pour ne jamais ouvrir plus d'une connexion par page.

MÉTHODES DÉTAILLÉES :

1. __construct() (Privé)
   - Rôle : Initialise l'objet `PDO` avec les paramètres du fichier `.env`. Privé pour empêcher les doublons de connexion.

2. getInstance() (Statique)
   - Rôle : Point d'accès unique à la BDD.
   ```php
   public static function getInstance() {
       if (self::$instance === null) {
           self::$instance = new PDO(...);
       }
       return self::$instance;
   }
   ```

PRINCIPE : LE SINGLETON
-----------------------
Le Singleton garantit qu'une seule et unique connexion à la base de données est ouverte pendant toute la durée du chargement d'une page. 
Cela économise des ressources serveur et centralise la configuration de sécurité. C'est un pilier de la performance du projet.
