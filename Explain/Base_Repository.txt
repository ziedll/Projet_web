CLASSE : BaseRepository (Base)
FICHIER : src/Repositories/BaseRepository.php

DESCRIPTION :
Classe abstraite regroupant les fonctionnalités communes à tous les accès aux données. Elle évite la répétition de code SQL simple.

MÉTHODES DÉTAILLÉES :

1. __construct()
   - Rôle : Récupère automatiquement l'objet PDO via `Database::getInstance()` et l'affecte à `$this->db` pour que les classes enfants puissent l'utiliser.

2. getAll()
   - Retour : Liste complète d'enregistrements (array).
   - Rôle : Récupère toutes les lignes de la table définie (ex: tous les articles) du plus récent au plus ancien.

3. getById($id)
   - Paramètre : `$id` (L'ID de l'élément en BDD).
   - Retour : Un tableau représentant la ligne trouvée ou `false`.
   - Rôle : Cherche un enregistrement précis via une requête préparée (sécurisée contre les injections SQL).

4. delete($id)
   - Rôle : Supprime définitivement une ligne de la table spécifiée par son identifiant.

NOTE TECHNIQUE :
La variable `$this->table` définie dans chaque repository enfant permet à ces fonctions de savoir sur quelle table SQL elles doivent agir.
