CLASSE : BaseRepository (Base)
FICHIER : src/Repositories/BaseRepository.php

DESCRIPTION :
Classe abstraite regroupant les fonctionnalités communes à tous les accès aux données. Elle évite la répétition de code SQL simple.

MÉTHODES DÉTAILLÉES :

1. __construct()
   - Rôle : Récupère automatiquement l'objet PDO via `Database::getInstance()` pour que les enfants l'utilisent via `$this->db`.

2. getAll() / getById($id) / delete($id)
   - Rôle : Fonctions CRUD génériques utilisables par n'importe quelle table SQL.
   ```php
   // Exemple générique utilisé par toutes les tables :
   $stmt = $this->db->prepare("SELECT * FROM {$this->table} WHERE id = ?");
   ```

PRINCIPE : L'ABSTRACTION DE DONNÉES
-----------------------------------
En regroupant les requêtes SQL les plus courantes dans cette classe de base, on réduit drastiquement les risques d'erreurs. 
Chaque Repository spécifique (User, Article, etc.) hérite de ces fonctions et n'a plus qu'à définir le nom de sa table (`protected $table`). 
Cela rend l'architecture extrêmement flexible et robuste.
