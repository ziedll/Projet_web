CONTRÔLEUR : ActualiteController
FICHIER : src/Controllers/ActualiteController.php

MÉTHODES DÉTAILLÉES :

1. index()
   - Rôle : Point d'entrée unique pour les actualités.
   - Logique : 
     - Si `$_GET['id']` est présent : Affiche un article seul (`actualite_detail.php`).
     - Sinon : Affiche la liste complète (`actualite.php`).
   ```php
   if ($id) {
       $article = $repo->getById($id);
       $this->render('actualite_detail', ['article' => $article]);
   } else {
       $articles = $repo->getAll();
       $this->render('actualite', ['articles' => $articles]);
   }
   ```

PRINCIPE : ROUTE DOUBLE USAGE
-----------------------------
Cette méthode utilise une seule route URL pour deux fonctionnalités différentes. Cela simplifie la maintenance et permet de partager la même logique de récupération de données (`ArticleRepository`) pour la liste et pour le détail.
C'est un exemple parfait de réutilisation de code via le contrôleur.
