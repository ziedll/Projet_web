CLASSE : Router (Core)
FICHIER : src/Core/Router.php

DESCRIPTION :
Analyse les URLs saisies (ex: `/boutique`) et les traduit en actions dans le code (Contrôleurs).

MÉTHODES DÉTAILLÉES :

1. add($route, $controller, $method)
   - Rôle : Remplit le dictionnaire des routes autorisées du site.

2. run()
   - Rôle : La méthode principale qui lance le site.
   - Technique : Utilise l'instanciation dynamique pour appeler le bon contrôleur.
   ```php
   $controller = new $controllerName(); // Magie de PHP
   $controller->$methodName(); // Appelle l'action demandée
   ```

PRINCIPE : LE ROUTAGE DYNAMIQUE
-------------------------------
Tout le site passe par un seul point d'entrée (`index.php`). Le Router compare l'URL saisie avec son dictionnaire interne. 
S'il trouve une correspondance, il "lance" le contrôleur associé. Cela permet d'avoir des URLs propres et sécurisées, cachant l'arborescence réelle des fichiers.
