CLASSE : Router (Core)
FICHIER : src/Core/Router.php

DESCRIPTION :
Analyse les URLs saisies par l'utilisateur et les traduit en actions dans le code (Contrôleurs).

MÉTHODES DÉTAILLÉES :

1. add($route, $controller, $method)
   - Paramètres :
     - `$route` : L'URL (ex: '/boutique').
     - `$controller` : Le nom du contrôleur à appeler (ex: 'BoutiqueController').
     - `$method` : La fonction interne à exécuter (ex: 'index').
   - Rôle : Remplit un tableau interne (`$routes`) qui sert de dictionnaire au projet.

2. run()
   - Rôle : La méthode principale appelée à chaque chargement de page.
   - Logique :
     - Récupère le paramètre `url` envoyé par le `.htaccess`.
     - Nettoie l'URL pour qu'elle ressemble toujours à `/nom-de-la-page`.
     - Cherche si cette URL existe dans le dictionnaire créé par `add()`.
     - Si oui : Instancie le contrôleur, vérifie que la méthode existe, et l'exécute.
     - Si non : Envoie un code d'erreur HTTP 404.

NOTE TECHNIQUE :
Le Router utilise la réflexion PHP dynamique pour instancier les classes (`new $controllerName()`), ce qui permet d'ajouter des pages simplement en écrivant une ligne dans `index.php`.
