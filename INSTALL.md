# Installation du Projet

## Commandes à lancer
1. `composer install` (Installe les librairies PHP)

## Configuration
1. Créer une base de données MySQL nommée `spider`.
2. Importer le schéma depuis `Explain/sql.txt`.
3. Vérifier le fichier `.env` à la racine.

## Pré-requis
- PHP 8.0+
- MySQL
- Composer
